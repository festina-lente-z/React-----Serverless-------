## 为什么要用 scss
因为scss提供的一些额外的语法会让我们更方便地去写样式代码。
要用scss首先要进行安装`npm install node-sass --save`
当引入`import './style.scss'`时，样式是全局生效的。
如果样式只是局部使用可以：
```javascript
import styles from './style.module.scss'

<div className={styles.app}>hello, Dell!</div>
```
